关于不同空间的一些思考

之前记录过自己对于空间变换的一些思考，这里再记一记关于不同空间的认识。

标量场

我们自从认识世界以来，较之向量，标量肯定出现得更早。人们使用标量无非记录各种数据，比如一个房间里面有各种家具，做家具的材料密度大多，桌子什么尺寸，室内多少温度，空气成分比例等等都可以用标量来表示，这个房间里充满了各种数字。

我们把这些数字集合起来构成了一个新的数学概念 A，通过研究发现 A 中的任何两个数字相加或者相乘之后，其结果依然是概念 A 中的标量，于是大家称呼概念 A 为标量场（Scalar Field）。

后来人们意识到有没有个标量加上自己或者乘以自己依然还是自己，这便引入了加法单位元 0 与乘法单位元 1 的概念。

向量空间（线性空间）

现在我们可以用一个标量场表示这个房间中的任何一个具体的数字，诸如上面所说的尺寸、密度、温度等。

现在你在房间外边，需要告诉另一个人去这个房间取出来一张桌子，你会怎么形容？

如果限制在标量场下，你会告诉他：房间里面有一个 50×40×23 的桌子，你帮忙拿出来。

倘若房间中恰巧有两张这个大小的桌子，你可能会额外加上木头密度为 0.5 的那一张，或者其他什么信息来表征你所想要的那张唯一的桌子。

问题是：如果你告诉他密度了，他怎么会马上知晓木头的密度？于是人们一片繁琐与混乱。慢慢地，向量产生了。

认知向量空间一定抓住向量的一些性质：只有方向与大小，没有位置。

向量空间是标量场的一个超集，不但有标量，还外加了由有限个标量组成的向量。

说到向量一定要说基，没有基的话，向量无从定义。基是由 n 个线性无关的向量组成的，二维向量空间 n = 2，三维就是 n = 3……

向量空间对向量的运算也有一定的定义：

  • 一个标量可以与一个向量相乘，其结果仍然是一个向量，其长度变为原来的标量倍；
  • 两个向量可以相加，其结果仍然是另一个向量。

就这两条（向量相减其实就是向量相加的一种），现在身处房外的你可以说：

给我拿来 50×40×23 的、头朝西的那张桌子。

明显是不是比之前好一些？

仿射空间

我想大家应该也想象到仿射空间扩展了什么，没错，就是点（Point）。

仿射空间定义了：

  • 一个点可以与一个向量相加，其结果是另一个点；
  • 两个点可以相减，其结果是一个向量。

仿射空间的到来使得人们不但可以让朋友进去拿东西，而且能够更加精确地拿，更能让他将桌子朝着某个方向移动多少个单位等等。

中学里我们见到了坐标系（二维或三维），可想而知它是在仿射空间下定义的，因为其中包含了点的概念，而点正是在仿射空间中引入的。

当然，中学里的坐标系也是一个特殊的仿射空间，因为其三个基向量彼此垂直（正交）。

欧几里得空间我没有细看，听说增加了“距离”的概念。我想，向量不是有大小吗？不能表示距离吗？哦，好复杂，就这样吧。

仿射加法

从上面得知，仿射空间中没有定义两个点相加会得到什么。

想一想也是，两个点相加到底应该得到另一个点、一个向量，还是一个标量呢？

就计算机图形学而言，我们在给两个点（假设它们定义了一条线段）进行插值时，其表达式形式就类似于两个点相加（我们强行认为结果仍然是另一个点）。

例如：

Q = P + n ⋅ v

表示点 P 沿着向量 v 的方向移动 n 个 v 的长度，到达点 Q。

由于：

v = Q - P

于是得到：

Q = P + n ( Q - P )

进一步整理可得：

Q = n Q + ( 1 - n ) P

这在形式上就好像两个仿射空间中的点进行了加法。

由于仿射空间本身并没有定义这种运算，人们便将这种形式称为仿射加法（Affine Combination）。

上面所言可能看起来有些不知所云，前提是对这些概念有一定了解。

它们都是我在认知数学概念过程中产生的各种想法，然后用比喻的方式试图解释这些概念为何会出现，纯属自己思维的认知历程。

还是那句话：错不要紧，及时改正就是了。
